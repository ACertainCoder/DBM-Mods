# My-DBM-Mods
These are my own mods for DBM:

- Call Command or Event

- Convert Variable
- Create Embed Message
- Find Item in List
- Loop Queue
- Play YouTube Playlist
- Restart Bot
- Set Embed Description
- Set Embed Footer
- Skip Queue
- Store Audio Info
- Store Server Info
- Store Twitch Info
- YouTube Search

Alternatives:

- Create Embed (Without Timestamp options)
- Set Embed Timestamp
- Store YouTube Playlist Info
- Store YouTube Info

## Warning
These mods have been made for DBM v1.5 and won't work with the current version anymore.
