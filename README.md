# DBM-Mods
These are my own mods for DBM:

- Call Command or Event
- Convert Time
- Convert Variable
- Create Embed Message
- Create Leaderboard
- Find Item in List
- Loop Queue
- Play YouTube Playlist
- Restart Bot
- Set Embed Description
- Set Embed Footer
- Skip Queue
- Store Audio Info
- Store Server Info
- Store Twitch Info
- YouTube Search

Alternatives:

- Create Embed (Without Timestamp options)
- Set Embed Timestamp
- Store YouTube Playlist Info
- Store YouTube Info

## Warning
These mods have been made for Discord Bot Maker v1.5 and may no longer work with the current version of DBM anymore.
